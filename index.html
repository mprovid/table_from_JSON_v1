Table, List, or Select Menu from JSON Data

The table has headers and scope. The titles of the headers and categories come from the JSON file. The data cells indicate if the intersection of the sets is true or false. The content in the data cells, the ✓ (checkmark) reads as "yes" in screen readers (the absence of a checkmark reads "no").

The advantage to using JSON depends on how many ways and how many places the data needs to be presented. While the entire data file can be represented as a table or a list, it also allows for only a portion of the data to be shared if that's what the context calls for.
